# Pakistan Utility Portal

This is a website for fetching electricity, water, and gas bills in Pakistan.
It also shows load shedding schedules, last meter reading, and power cut reasons.

## Features
- Fetch duplicate bills (Electricity / Water / Gas)
- Load shedding timings per area
- Power cut reasons
- Last meter reading

## How to Use
1. Open `generate-bill.html` to fetch bills
2. Open `load-shedding.html` to see outages and alerts
